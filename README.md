# Archisight

**Archisight** (Archive + Sight) — это легковесная утилита для Windows, предназначенная для автоматического мониторинга папки-архива и очистки устаревших данных. Программа сканирует целевую директорию, находит подпапки, названные по датам (например, `10.07.26`), и автоматически удаляет старые архивы целиком, оставляя данные только за последние N дней.

---

## ⚙️ Принцип работы и алгоритм очистки

Логика удаления спроектирована так, чтобы гарантировать безопасность пользовательских файлов и удалять только целевые архивы:

1. **Глубина сканирования:** Анализируются **только** подпапки первого уровня внутри выбранного пути к архиву. Программа не сканирует вложенные директории.
2. **Строгое соответствие имени:** Обрабатываются только те папки, чьё имя *целиком* совпадает с заданным в настройках форматом даты (порядок день/месяц/год, разделитель, короткий или полный формат года). Короткий год (например, `26`) трактуется как текущее столетие (`2026`).
3. **Поиск "опорной точки":** Среди распознанных дат определяется самая свежая, которая не превышает текущую системную дату.
4. **Граница хранения:** От "опорной точки" отсчитывается назад заданное количество *календарных* дней. 
5. **Удаление:** Все распознанные папки с датой в пределах границы — сохраняются. Все папки, дата которых старше — безвозвратно удаляются вместе со всем содержимым.
6. **Пропуски дат:** Отсутствие папок за определенные дни (например, в выходные) не нарушает расчет.
7. **Безопасность:** Любые сторонние файлы или папки, не подпадающие под формат даты, **игнорируются** и никогда не удаляются. Их список можно посмотреть в разделе "Показать сводку".

## 🚀 Быстрый старт (Запуск из исходников)

Для запуска программы без сборки потребуется **Python 3.10+**. 
*Примечание: при установке Python для Windows обязательно убедитесь, что включена опция **"tcl/tk and IDLE"** (необходима для работы графического интерфейса).*

1. Склонируйте репозиторий:
   ```bash
   git clone [https://github.com/your-username/archisight.git](https://github.com/your-username/archisight.git)
   cd archisight

```

2. Установите зависимости:
```bash
pip install -r requirements.txt

```


3. Запустите приложение:
```bash
python run_archisight.py

```



---

## 📦 Сборка в автономный .exe файл

Проект настроен для быстрой сборки в один самодостаточный исполняемый файл, который можно переносить на другие ПК без установки Python.

1. Запустите скрипт сборки:
```cmd
build\build.bat

```


2. Готовый файл `Archisight.exe` (со встроенной иконкой) появится в папке `dist\`.

> **Важно:** При сборке в один файл (`--onefile`) встроенные защитники Windows иногда могут выдавать ложные срабатывания (False Positive). Если это мешает работе, откройте файл `build\build.bat` и замените флаг `--onefile` на `--onedir`. В этом случае программа соберется в папку `dist\Archisight\`, что практически исключает конфликты с антивирусами.

## 🧪 Тестирование

Вся логика сканирования и расчета дат вынесена в независимый модуль (`archisight/core.py`), который полностью отвязан от GUI и покрыт Unit-тестами.

Для проверки работоспособности алгоритма выполните команду:

```bash
python -m unittest discover tests -v

```

Тесты имитируют различные сценарии (включая пропуски дат и смену форматов), гарантируя, что программа оставит ровно нужное количество папок.

## 🖥 Особенности интерфейса и интеграции

* **Автозапуск:** Реализован через стандартный пользовательский ключ реестра `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`. Права администратора не требуются. При старте вместе с системой программа запускается сразу в свернутом виде (в трее).
* **Синхронизация статуса:** Статус "Активно/Неактивно" на главном экране отражает *реальное* состояние ключа в реестре Windows, а не просто сохраненную галочку.
* **Поведение в трее:** Иконка в системном трее появляется только тогда, когда окно закрыто крестиком при включенном автозапуске (программа переходит в фоновый режим). При разворачивании окна иконка скрывается, чтобы не дублировать активное приложение.
* **Безопасная настройка ("Сбросить" и "Сохранить"):** Кнопка "Сбросить" возвращает поля к значениям по умолчанию (хранение 90 дней, автозапуск выключен), но *не вносит* изменений в систему до нажатия "Сохранить".
* **Применить сейчас:** Кнопка для ручного и немедленного прогона алгоритма очистки с использованием тех настроек, которые в данный момент введены в окне (даже если они еще не сохранены). Требует подтверждения, так как процесс удаления необратим.
* **Хранение настроек:** Файл конфигурации автоматически создается и хранится по пути: `%APPDATA%\Archisight\config.json`.

## 📂 Структура проекта

```text
archisight/
├── run_archisight.py         # Точка входа приложения
├── requirements.txt          # Зависимости Python
├── build/
│   └── build.bat             # Скрипт автоматической сборки в .exe
├── tests/
│   └── test_core.py          # Автотесты логики удаления
└── archisight/
    ├── core.py               # Ядро: логика сканирования и математика дат
    ├── config.py             # Сохранение/загрузка настроек (config.json)
    ├── i18n.py               # Система локализации (RU/EN)
    ├── autostart.py          # Взаимодействие с реестром Windows
    ├── scheduler.py          # Фоновый планировщик ежедневной проверки
    ├── tray.py               # Логика работы системного трея (pystray)
    ├── main.py               # Связующий модуль приложения
    ├── resources.py          # Работа с путями и ассетами
    ├── assets/               # Иконки (icon.ico, icon.png)
    └── gui/                  # Графический интерфейс на Tkinter
        ├── style.py          # Цветовая палитра и стили
        ├── main_window.py    # Главный экран
        ├── settings_window.py # Окно настроек
        └── about_window.py   # Окно "О программе"

```
